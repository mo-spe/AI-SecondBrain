# 图标占位符说明

此目录需要包含以下图标文件：

## 必需的图标文件

1. **icon16.png** (16x16 像素)
   - 用于浏览器工具栏小图标
   - 建议设计：简化的脑部图标

2. **icon48.png** (48x48 像素)
   - 用于扩展管理页面
   - 建议设计：清晰的脑部图标

3. **icon128.png** (128x128 像素)
   - 用于 Chrome Web Store
   - 建议设计：详细的脑部图标

## 设计建议

### 颜色方案
- 主色：#667eea (蓝色)
- 辅色：#764ba2 (紫色)
- 背景：白色或透明

### 图标元素
- 脑部轮廓或简化的大脑图标
- 可选：添加知识符号（书本、灯泡等）
- 风格：现代、扁平化设计

### 创建方法

#### 方法 1：使用在线工具
1. 访问 https://www.favicon-generator.org/
2. 上传设计好的图标
3. 下载不同尺寸的 PNG 文件

#### 方法 2：使用设计软件
1. 使用 Figma、Sketch 或 Adobe XD
2. 设计图标（建议 512x512）
3. 导出为 PNG 格式
4. 使用工具调整尺寸

#### 方法 3：使用 AI 生成
1. 使用 Midjourney、DALL-E 或其他 AI 绘图工具
2. 提示词："modern brain icon, flat design, blue and purple gradient, minimal"
3. 下载生成的图片
4. 调整尺寸并导出

## 临时解决方案

在添加正式图标之前，可以使用以下方法：

### 选项 1：使用 emoji 作为图标
修改 `manifest.json`：
```json
"default_icon": {
  "16": "icons/icon16.png",
  "48": "icons/icon48.png",
  "128": "icons/icon128.png"
}
```

暂时使用简单的 SVG 图标或在线图标。

### 选项 2：使用占位符图标
可以使用任何 16x16、48x48、128x128 的 PNG 图片作为占位符。

## 注意事项

1. 确保图标背景透明（PNG 格式）
2. 图标应该在小尺寸下仍然清晰可见
3. 避免使用过多细节，保持简洁
4. 确保图标在深色和浅色背景下都清晰可见
