## 🎯 核心部署文件

### 1. docker-compose.prod.yml ✓
**路径**: `d:\AI-SecondBrain\docker-compose.prod.yml`

**说明**: 生产环境 Docker 编排文件

**包含服务**:
- MySQL 8.0（数据库）
- Redis 7（缓存）
- Zookeeper + Kafka（消息队列）
- Elasticsearch 8.11（搜索引擎）
- DeerFlow（AI 服务）
- Spring Boot 后端
- Vue.js 前端（构建阶段）
- Nginx（Web 服务器）

**主要特性**:
- ✅ 健康检查配置
- ✅ 数据持久化
- ✅ 网络隔离
- ✅ 环境变量注入
- ✅ 资源限制
- ✅ 日志管理

**使用方法**:
```bash
# 构建并启动
docker-compose -f docker-compose.prod.yml up -d

# 查看状态
docker-compose -f docker-compose.prod.yml ps

# 查看日志
docker-compose -f docker-compose.prod.yml logs -f
```

## 🔒 安全注意事项

### 敏感文件保护
以下文件包含敏感信息，不应提交到 Git：
- `.env.production`
- `ssl/cert.pem`
- `ssl/key.pem`
- `logs/`

### 密码安全
- 使用强密码（至少 16 位）
- 包含大小写字母、数字、特殊字符
- 定期更换密码

### 网络安全
- 配置防火墙
- 只开放必要端口
- 使用 SSH 密钥登录
- 禁用 root 密码登录
