# AI-SecondBrain 安装包说明

## 📦 安装包内容

本安装包包含AI-SecondBrain项目的完整部署文件，支持一键部署到服务器。

### 文件列表

```
AI-SecondBrain-安装包/
├── README.txt                    # 安装说明
├── deploy.sh                     # Linux一键部署脚本
├── deploy.ps1                    # Windows一键部署脚本
├── docker-compose.yml            # Docker编排文件
├── .env.example                  # 环境配置模板
├── Dockerfile                    # 后端Dockerfile
├── frontend/
│   └── Dockerfile               # 前端Dockerfile
├── nginx/
│   ├── nginx.conf               # Nginx主配置
│   └── conf.d/
│       └── default.conf         # Nginx站点配置
├── init_database.sql            # 数据库初始化脚本
├── manage.sh                    # 服务管理脚本
└── DEPLOYMENT.md                # 详细部署文档
```

## 🚀 快速部署指南

### 前置要求

- 服务器：2核CPU、4GB内存、20GB硬盘
- 操作系统：Ubuntu 20.04+ / CentOS 7+ / Windows Server 2019+
- 已安装Docker和Docker Compose

### 部署步骤

#### Linux系统

```bash
# 1. 解压安装包
unzip AI-SecondBrain-安装包.zip
cd AI-SecondBrain-安装包

# 2. 安装Docker（如果未安装）
curl -fsSL https://get.docker.com | bash
curl -L "https://github.com/docker/compose/releases/download/v2.23.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
chmod +x /usr/local/bin/docker-compose

# 3. 运行部署脚本
chmod +x deploy.sh
./deploy.sh

# 4. 按提示配置环境变量
# 必填项：
# - MYSQL_ROOT_PASSWORD: MySQL root密码
# - REDIS_PASSWORD: Redis密码
# - DEEPSEEK_API_KEY: DeepSeek API密钥
# - DOMAIN_NAME: 你的域名（可选）
```

#### Windows系统

```powershell
# 1. 解压安装包
Expand-Archive AI-SecondBrain-安装包.zip -DestinationPath .
cd AI-SecondBrain-安装包

# 2. 安装Docker Desktop（如果未安装）
# 下载地址：https://www.docker.com/products/docker-desktop

# 3. 以管理员身份运行PowerShell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# 4. 运行部署脚本
.\deploy.ps1

# 5. 按提示配置环境变量
```

## 📝 配置说明

### 必填配置项

编辑 `.env` 文件，填写以下配置：

```env
# 数据库配置
MYSQL_ROOT_PASSWORD=your_secure_password
MYSQL_PASSWORD=your_secure_password

# Redis配置
REDIS_PASSWORD=your_redis_password

# AI服务配置（至少配置一个）
DEEPSEEK_API_KEY=your_deepseek_api_key

# 域名配置（可选）
DOMAIN_NAME=your-domain.com
```

### 可选配置项

```env
# 端口配置
BACKEND_PORT=8080
FRONTEND_PORT=80

# JVM内存配置
JAVA_OPTS=-Xms512m -Xmx1024m

# Elasticsearch内存配置
ES_JAVA_OPTS=-Xms512m -Xmx512m
```

## ❓ 常见问题

### 1. Docker未安装

**解决方案**：按照上面的步骤安装Docker和Docker Compose

### 2. 端口被占用

**解决方案**：修改 `.env` 文件中的端口配置

### 3. 内存不足

**解决方案**：调整 `.env` 文件中的内存配置，或升级服务器配置

### 4. 无法访问

**解决方案**：
- 检查防火墙设置
- 检查云服务器安全组规则
- 确认服务已正常启动
