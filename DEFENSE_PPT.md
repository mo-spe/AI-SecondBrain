### 演讲脚本（50 秒）
> 系统采用经典的分层架构设计，自顶向下分为四层：
> 
> 第一层是客户端层：
> 包含 Web 应用（基于 Vue 3）、浏览器插件（Chrome 扩展），以及未来规划的移动端 App。
> 
> 第二层是 API 网关层：
> 使用 Nginx 作为反向代理服务器，负责负载均衡和静态资源缓存。

> 包含两个核心服务——Spring Boot 后端（Java 21）负责业务逻辑，DeerFlow AI 服务（Python FastAPI）负责 AI 处理。
> 
> 第四层是数据层：
> 使用 MySQL 存储结构化数据，Redis 做缓存，Kafka 处理异步任务，Elasticsearch 提供全文检索。
> 
> 这种分层架构职责清晰，便于开发、测试和部署。
