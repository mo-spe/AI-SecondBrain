# AI-SecondBrain 安装及使用指南

## 1. 环境要求

### 1.1 硬件要求
- **CPU**: 4核及以上
- **内存**: 8GB及以上（推荐16GB）
- **硬盘**: 50GB及以上可用空间
- **网络**: 稳定的互联网连接

### 1.2 软件要求

#### 后端环境
- **JDK**: 21或更高版本
- **Maven**: 3.8或更高版本
- **MySQL**: 8.0或更高版本
- **Redis**: 7或更高版本
- **Kafka**: 3.6或更高版本
- **Elasticsearch**: 8.11或更高版本

#### 前端环境
- **Node.js**: 18或更高版本
- **npm**: 9或更高版本

#### 浏览器插件
- **Chrome**: 120或更高版本
- **Edge**: 120或更高版本

## 2. 安装步骤

### 2.1 数据库安装

#### MySQL安装

**Windows系统**：
1. 下载MySQL安装包：https://dev.mysql.com/downloads/mysql/
2. 运行安装程序，按照向导完成安装
3. 设置root密码（建议：root123456）
4. 启动MySQL服务

**Linux系统**：
```bash
sudo apt update
sudo apt install mysql-server
sudo systemctl start mysql
sudo mysql_secure_installation
```

**创建数据库**：
```sql
CREATE DATABASE second_brain CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### Redis安装

**Windows系统**：
1. 下载Redis for Windows：https://github.com/microsoftarchive/redis/releases
2. 解压到指定目录
3. 运行redis-server.exe

**Linux系统**：
```bash
sudo apt install redis-server
sudo systemctl start redis
```

#### Kafka安装

**Windows系统**：
1. 下载Kafka：https://kafka.apache.org/downloads
2. 解压到指定目录
3. 修改config/server.properties
4. 运行bin/windows/zookeeper-server-start.bat
5. 运行bin/windows/kafka-server-start.bat

**Linux系统**：
```bash
sudo apt install kafka
sudo systemctl start zookeeper
sudo systemctl start kafka
```

#### Elasticsearch安装

**Windows系统**：
1. 下载Elasticsearch：https://www.elastic.co/downloads/elasticsearch
2. 解压到指定目录
3. 修改config/elasticsearch.yml
4. 运行bin/elasticsearch.bat

**Linux系统**：
```bash
wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-8.11.0-linux-x86_64.tar.gz
tar -xzf elasticsearch-8.11.0-linux-x86_64.tar.gz
cd elasticsearch-8.11.0
./bin/elasticsearch
```

### 2.2 后端安装

#### 1. 克隆项目
```bash
git clone https://github.com/yourusername/AI-SecondBrain.git
cd AI-SecondBrain
```

#### 2. 配置数据库连接

编辑`src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/second_brain?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: root
    password: root123456
  redis:
    host: localhost
    port: 6379
  kafka:
    bootstrap-servers: localhost:9092
  elasticsearch:
    uris: http://localhost:9200
```

#### 3. 初始化数据库

**方法一：使用PowerShell脚本（推荐）**
```powershell
.\init_database.ps1
```

**方法二：手动执行SQL**
```bash
mysql -u root -p second_brain < init_database.sql
```

#### 4. 编译项目
```bash
mvn clean compile
```

#### 5. 启动后端服务
```bash
mvn spring-boot:run
```

后端服务将在 http://localhost:8080 启动

### 2.3 前端安装

#### 1. 进入前端目录
```bash
cd frontend
```

#### 2. 安装依赖
```bash
npm install
```

#### 3. 配置后端地址

编辑`frontend/src/utils/request.js`：

```javascript
const baseURL = 'http://localhost:8080/api';
```

#### 4. 启动前端服务
```bash
npm run dev
```

前端服务将在 http://localhost:3000 启动

### 2.4 浏览器插件安装

#### Chrome浏览器

1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `extension` 文件夹
6. 插件安装成功！

#### Edge浏览器

1. 打开Edge浏览器
2. 访问 `edge://extensions/`
3. 开启"开发人员模式"
4. 点击"加载解压缩的扩展"
5. 选择 `extension` 文件夹
6. 插件安装成功！

## 3. 使用指南

### 3.1 用户注册

1. 访问 http://localhost:3000
2. 点击"注册"按钮
3. 填写用户名、密码、邮箱等信息
4. 点击"注册"按钮
5. 注册成功后自动登录

### 3.2 浏览器插件使用

#### 首次使用

1. 点击浏览器工具栏中的插件图标
2. 在弹窗中输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后，状态会显示"已登录"

#### 采集对话

1. 访问支持的AI对话网站（ChatGPT/DeepSeek/Kimi）
2. 进行一段有价值的对话
3. 点击页面右上角的"采集到知识库"按钮
4. 等待采集完成提示
5. 知识点会自动保存到您的知识库

### 3.3 知识管理

#### 查看知识点列表

1. 登录后，点击"知识管理"菜单
2. 查看所有知识点列表
3. 可以按重要程度、掌握程度等筛选

#### 查看知识点详情

1. 在知识点列表中，点击"查看详情"按钮
2. 查看知识点的完整内容
3. 查看复习进度和统计信息

#### 编辑知识点

1. 在知识点详情页面，点击"编辑"按钮
2. 修改知识点内容
3. 点击"保存"按钮

#### 删除知识点

1. 在知识点列表中，点击"删除"按钮
2. 确认删除操作

#### 导出知识点

1. 勾选要导出的知识点
2. 选择导出格式（Markdown/PDF/Word/JSON/CSV）
3. 点击"导出"按钮
4. 文件自动下载

### 3.4 复习中心

#### 查看待复习卡片

1. 点击"复习中心"菜单
2. 查看"今日待复习"列表
3. 查看复习统计信息

#### 开始复习

1. 在待复习列表中，点击"开始复习"按钮
2. 查看复习题目
3. 选择答案
4. 点击"提交答案"按钮

#### 查看复习结果

1. 提交答案后，查看复习结果
2. 查看正确答案和解析
3. 查看本次复习统计

#### 生成复习题目

1. 在知识管理页面，找到要复习的知识点
2. 点击"生成题目"按钮
3. 等待题目生成完成
4. 题目会自动添加到复习中心

### 3.5 搜索功能

## 5. 常见问题

### 5.1 数据库连接失败

**问题**: 启动后端时报错"Could not connect to database"

**解决方案**:
1. 检查MySQL服务是否启动
2. 检查数据库连接配置是否正确
3. 检查防火墙是否允许连接

### 5.2 Redis连接失败

**问题**: 启动后端时报错"Could not connect to Redis"

**解决方案**:
1. 检查Redis服务是否启动
2. 检查Redis连接配置是否正确
3. 检查Redis端口是否被占用

### 5.3 Kafka连接失败

**问题**: 启动后端时报错"Could not connect to Kafka"

**解决方案**:
1. 检查Kafka服务是否启动
2. 检查Zookeeper服务是否启动
3. 检查Kafka连接配置是否正确

### 5.4 Elasticsearch连接失败

**问题**: 启动后端时报错"Could not connect to Elasticsearch"

**解决方案**:
1. 检查Elasticsearch服务是否启动
2. 检查Elasticsearch连接配置是否正确
3. 检查Elasticsearch索引是否创建

### 5.5 浏览器插件无法采集

**问题**: 点击"采集"按钮没有反应

**解决方案**:
1. 检查后端服务是否启动
2. 检查插件是否已登录
3. 检查网络连接是否正常
4. 检查浏览器控制台是否有错误信息

### 5.6 前端无法访问后端

**问题**: 前端页面报错"Network Error"

**解决方案**:
1. 检查后端服务是否启动
2. 检查后端地址配置是否正确
3. 检查CORS配置是否正确
