### 1.2 数据表设计

#### 1.2.1 用户表 (user)

| 字段名          | 类型     | 长度 | 允许NULL | 默认值                                        | 说明                               |
| --------------- | -------- | ---- | -------- | --------------------------------------------- | ---------------------------------- |
| id              | BIGINT   | -    | NO       | AUTO_INCREMENT                                | 主键ID                             |
| username        | VARCHAR  | 50   | NO       | -                                             | 用户名，唯一                       |
| password        | VARCHAR  | 255  | NO       | -                                             | 密码（BCrypt加密）                 |
| email           | VARCHAR  | 100  | YES      | NULL                                          | 邮箱                               |
| phone           | VARCHAR  | 20   | YES      | NULL                                          | 手机号                             |
| bio             | VARCHAR  | 500  | YES      | NULL                                          | 个人简介                           |
| avatar          | VARCHAR  | 500  | YES      | NULL                                          | 头像URL                            |
| register_time   | DATETIME | -    | NO       | CURRENT_TIMESTAMP                             | 注册时间                           |
| last_login_time | DATETIME | -    | YES      | NULL                                          | 最后登录时间                       |
| create_time     | DATETIME | -    | NO       | CURRENT_TIMESTAMP                             | 创建时间                           |
| update_time     | DATETIME | -    | NO       | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间                           |
| deleted         | TINYINT  | -    | NO       | 0                                             | 逻辑删除标识（0-未删除，1-已删除） |

**索引设计**：

- PRIMARY KEY (id)
- UNIQUE KEY uk_username (username)
- KEY idx_email (email)
- KEY idx_create_time (create_time)

#### 1.2.2 原始对话记录表 (raw_chat_record)

| 字段名      | 类型     | 长度 | 允许NULL | 默认值                                        | 说明                                    |
| ----------- | -------- | ---- | -------- | --------------------------------------------- | --------------------------------------- |
| id          | BIGINT   | -    | NO       | AUTO_INCREMENT                                | 主键ID                                  |
| user_id     | BIGINT   | -    | NO       | -                                             | 用户ID                                  |
| platform    | VARCHAR  | 50   | YES      | NULL                                          | 来源平台（ChatGPT/DeepSeek/Kimi/Other） |
| content     | TEXT     | -    | NO       | -                                             | 对话内容                                |
| source_url  | VARCHAR  | 500  | YES      | NULL                                          | 原始链接                                |
| create_time | DATETIME | -    | NO       | CURRENT_TIMESTAMP                             | 创建时间                                |
| update_time | DATETIME | -    | NO       | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间                                |
| deleted     | TINYINT  | -    | NO       | 0                                             | 逻辑删除标识                            |

**索引设计**：

- PRIMARY KEY (id)
- KEY idx_user_id (user_id)
- KEY idx_platform (platform)
- KEY idx_create_time (create_time)

#### 1.2.3 知识节点表 (knowledge_node)

| 字段名           | 类型     | 长度 | 允许NULL | 默认值                                        | 说明                        |
| ---------------- | -------- | ---- | -------- | --------------------------------------------- | --------------------------- |
| id               | BIGINT   | -    | NO       | AUTO_INCREMENT                                | 主键ID                      |
| user_id          | BIGINT   | -    | NO       | -                                             | 用户ID                      |
| chat_record_id   | BIGINT   | -    | YES      | NULL                                          | 来源对话记录ID              |
| title            | VARCHAR  | 200  | NO       | -                                             | 知识点标题                  |
| content_md       | TEXT     | -    | YES      | NULL                                          | Markdown格式内容            |
| summary          | TEXT     | -    | YES      | NULL                                          | 摘要                        |
| vector_id        | VARCHAR  | 100  | YES      | NULL                                          | 向量ID（用于Elasticsearch） |
| importance       | TINYINT  | -    | NO       | 3                                             | 重要程度（1-5）             |
| mastery_level    | TINYINT  | -    | NO       | 0                                             | 掌握程度（0-5）             |
| review_count     | INT      | -    | NO       | 0                                             | 复习次数                    |
| create_time      | DATETIME | -    | NO       | CURRENT_TIMESTAMP                             | 创建时间                    |
| update_time      | DATETIME | -    | NO       | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间                    |
| last_review_time | DATETIME | -    | YES      | NULL                                          | 上次复习时间                |
| next_review_time | DATETIME | -    | YES      | NULL                                          | 下次复习时间                |
| deleted          | TINYINT  | -    | NO       | 0                                             | 逻辑删除标识                |

**索引设计**：

- PRIMARY KEY (id)
- KEY idx_user_id (user_id)
- KEY idx_chat_record_id (chat_record_id)
- KEY idx_next_review_time (next_review_time)

## 2. API接口设计

### 2.1 认证接口

#### 2.1.1 用户注册

**接口路径**：POST /api/auth/register

**请求参数**：

```json
{
  "username": "testuser",
  "password": "123456",
  "email": "test@example.com"
}
```

**响应结果**：

```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "userId": 1,
    "username": "testuser",
    "email": "test@example.com"
  }
}
```

#### 2.1.2 用户登录

**接口路径**：POST /api/auth/login

**请求参数**：

```json
{
  "username": "testuser",
  "password": "123456"
}
```

**响应结果**：

```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "userId": 1,
    "username": "testuser",
    "avatar": "https://example.com/avatar.jpg"
  }
}
```

### 2.2 知识管理接口

#### 2.2.1 获取知识列表

**接口路径**：GET /api/knowledge/list

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| current | Integer | 否 | 当前页，默认1 |
| size | Integer | 否 | 每页大小，默认10 |
| keyword | String | 否 | 搜索关键词 |
| importance | Integer | 否 | 重要程度（1-5） |
| masteryLevel | Integer | 否 | 掌握程度（0-5） |

**响应结果**：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "records": [
      {
        "id": 1,
        "title": "机器学习基础",
        "summary": "机器学习的基本概念和算法",
        "importance": 4,
        "masteryLevel": 3,
        "reviewCount": 5,
        "nextReviewTime": "2026-03-15 10:00:00"
      }
    ],
    "total": 100,
    "current": 1,
    "size": 10
  }
}
```

}

## 3. 核心算法设计

### 3.1 艾宾浩斯遗忘曲线算法

#### 3.1.1 算法描述

基于艾宾浩斯遗忘曲线的智能复习算法，根据用户的复习表现动态调整复习间隔。

#### 3.1.2 复习间隔设计

```java
private static final long[] REVIEW_INTERVALS = {
    1440,     // 第一次：24小时后（1440分钟）
    10080,    // 第二次：7天后（10080分钟）
    20160,    // 第三次：14天后（20160分钟）
    43200,    // 第四次：30天后（43200分钟）
    86400     // 第五次：60天后（86400分钟）
};
```

#### 3.1.3 算法实现

```java
@Override
public LocalDateTime calculateNextReviewTime(LocalDateTime lastReviewTime, int reviewCount, boolean isCorrect) {
    long intervalMinutes = calculateNextReviewInterval(reviewCount, isCorrect);
    LocalDateTime nextReviewTime = lastReviewTime.plusMinutes(intervalMinutes);
    return nextReviewTime;
}

private long calculateNextReviewInterval(int reviewCount, boolean isCorrect) {
    if (reviewCount < 0) {
        reviewCount = 0;
    }

    int intervalIndex = Math.min(reviewCount, REVIEW_INTERVALS.length - 1);
    long baseInterval = REVIEW_INTERVALS[intervalIndex];

    if (!isCorrect) {
        baseInterval = baseInterval / 2;
    }

    return baseInterval;
}
```

#### 3.1.4 掌握程度计算

```java
@Override
public int calculateMasteryLevel(int reviewCount, double accuracy) {
    double masteryScore = 0.0;

    masteryScore += Math.min(reviewCount * 10, 50);
    masteryScore += accuracy * 50;

    int masteryLevel = (int) Math.round(masteryScore / 10);

    return Math.max(0, Math.min(5, masteryLevel));
}
```

### 3.2 重要程度智能计算算法

#### 3.2.1 算法描述

多维度计算知识点重要程度，综合考虑手动标注、内容复杂度、复习频率等因素。

#### 3.2.2 权重设计

| 因素         | 权重 | 说明                         |
| ------------ | ---- | ---------------------------- |
| 手动重要程度 | 40%  | 用户手动标注的重要程度       |
| 内容复杂度   | 30%  | 基于内容长度和关键词数量计算 |
| 复习频率     | 20%  | 基于复习次数计算             |
| 交互频率     | 10%  | 基于用户交互次数计算         |

#### 3.2.3 算法实现

```java
@Override
public int calculateImportance(int manualImportance, int contentComplexity, int reviewFrequency, int interactionFrequency) {
    double importanceScore = 0.0;

    importanceScore += manualImportance * 0.4;
    importanceScore += contentComplexity * 0.3;
    importanceScore += reviewFrequency * 0.2;
    importanceScore += interactionFrequency * 0.1;

    int finalImportance = (int) Math.round(importanceScore);

    return Math.max(1, Math.min(5, finalImportance));
}

private int calculateContentComplexity(KnowledgeNode node) {
    int complexity = 3;

    String content = node.getContentMd() != null ? node.getContentMd() : "";

    int contentLength = content.length();

    if (contentLength > 2000) {
        complexity = 5;
    } else if (contentLength > 1000) {
        complexity = 4;
    } else if (contentLength > 500) {
        complexity = 3;
    } else if (contentLength > 200) {
        complexity = 2;
    } else {
        complexity = 1;
    }

    int keywordCount = countKeywords(content);
    if (keywordCount > 20) {
        complexity = Math.min(complexity + 1, 5);
    } else if (keywordCount < 5) {
        complexity = Math.max(complexity - 1, 1);
    }

    return complexity;
}
```

### 3.3 向量嵌入算法

#### 3.3.1 算法描述

使用阿里云text-embedding-v2模型生成文本向量，用于语义搜索。

#### 3.3.2 算法实现

```java
@Override
public List<Float> generateEmbedding(String text, String model) {
    String processedText = preprocessText(text);

    JSONObject requestBody = new JSONObject();
    requestBody.put("model", model);

    JSONObject input = new JSONObject();
    JSONArray texts = new JSONArray();
    texts.add(processedText);
    input.put("texts", texts);
    requestBody.put("input", input);

    Request request = new Request.Builder()
            .url(apiUrl)
            .addHeader("Authorization", "Bearer " + apiKey)
            .addHeader("Content-Type", "application/json")
            .post(RequestBody.create(requestBody.toJSONString(), MediaType.parse("application/json")))
            .build();

    try (Response response = client.newCall(request).execute()) {
        if (!response.isSuccessful()) {
            return new ArrayList<>();
        }

        String responseBody = response.body().string();
        JSONObject jsonResponse = JSON.parseObject(responseBody);

        JSONObject output = jsonResponse.getJSONObject("output");
        if (output != null) {
            JSONArray embeddings = output.getJSONArray("embeddings");
            if (embeddings != null && !embeddings.isEmpty()) {
                JSONObject embeddingObj = embeddings.getJSONObject(0);
                JSONArray embeddingArray = embeddingObj.getJSONArray("embedding");

                List<Float> embedding = new ArrayList<>();
                for (int i = 0; i < embeddingArray.size(); i++) {
                    embedding.add(embeddingArray.getFloatValue(i));
                }

                return embedding;
            }
        }
    } catch (IOException e) {
        log.error("生成Embedding失败", e);
    }

    return new ArrayList<>();
}
```

### 3.4 语义搜索算法

#### 3.4.1 算法描述

基于向量相似度的语义搜索，使用余弦相似度计算文本相似度。

#### 3.4.2 算法实现

```java
@Override
public List<KnowledgeNodeVO> semanticSearch(String queryText, Long userId, int topK) {
    List<Float> queryEmbedding = embeddingService.generateEmbedding(queryText);

    if (queryEmbedding.isEmpty()) {
        return new ArrayList<>();
    }

    SearchRequest searchRequest = SearchRequest.of(s -> s
            .index("knowledge_index")
            .query(q -> q
                    .bool(b -> b
                            .must(m -> m.term(t -> t.field("userId").value(userId)))
                            .filter(f -> f.exists(e -> e.field("vector")))
                    )
            )
            .knn(k -> k
                    .field("vector")
                    .queryVector(queryEmbedding)
                    .k(topK)
                    .numCandidates(100)
            )
            .size(topK)
    );

    try {
        SearchResponse<KnowledgeNode> response = elasticsearchClient.search(searchRequest, KnowledgeNode.class);

        return response.hits().hits().stream()
                .map(hit -> {
                    KnowledgeNode node = hit.source();
                    KnowledgeNodeVO vo = new KnowledgeNodeVO();
                    BeanUtils.copyProperties(node, vo);
                    vo.setScore(hit.score());
                    return vo;
                })
                .collect(Collectors.toList());
    } catch (IOException e) {
        log.error("语义搜索失败", e);
        return new ArrayList<>();
    }
}
```
