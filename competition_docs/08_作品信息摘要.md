## 作品简介

AI-SecondBrain是一个基于AI的智能知识管理系统，旨在帮助用户从AI对话中提取知识，并通过科学的复习算法实现知识的长期记忆。项目采用前后端分离架构，包含Web应用、浏览器插件和AI服务三个主要部分，实现了对话采集、知识提取、知识管理、智能复习、全文搜索等核心功能。

### 核心功能

1. **对话采集**：支持ChatGPT、DeepSeek、Kimi等主流AI平台，浏览器插件一键采集
2. **知识提取**：AI自动从对话中提取结构化知识点
3. **知识管理**：查看、编辑、删除知识点，支持批量导出
4. **智能复习**：基于艾宾浩斯遗忘曲线的复习提醒，自动生成复习题目
5. **知识搜索**：基于Elasticsearch的全文搜索，支持关键词高亮
6. **数据统计**：可视化的学习数据统计，包括复习完成率、掌握程度等

### 技术创新

1. **AI驱动的知识提取**：利用大语言模型自动从对话中提取结构化知识
2. **科学复习算法**：基于艾宾浩斯遗忘曲线的智能复习系统，动态调整复习间隔
3. **无缝集成体验**：浏览器插件实现与AI平台的深度集成，一键采集
4. **分布式架构**：采用Kafka消息队列实现异步处理，提升系统性能
5. **智能搜索**：结合Elasticsearch实现高效的全文搜索

   - 存储用户的复习记录

## 核心算法

### 1. 艾宾浩斯遗忘曲线算法

**复习间隔**：
- 第一次复习：24小时后
- 第二次复习：7天后
- 第三次复习：14天后
- 第四次复习：30天后
- 第五次复习：60天后

**动态调整**：
- 回答正确：按照标准间隔进行下次复习
- 回答错误：间隔缩短为标准间隔的一半

### 2. AI题目生成算法

**生成流程**：
1. 分析知识点内容，提取关键信息
2. 根据内容类型选择题目类型
3. 生成题目和选项
4. 评估题目质量
5. 计算题目难度

### 3. 连续学习天数计算

**算法**：从今天开始向前遍历，检查每天是否有复习记录，计算连续学习天数。
